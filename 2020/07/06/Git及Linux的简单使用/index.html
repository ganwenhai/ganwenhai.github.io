Git及Linux的简单使用

发表于 2020-07-06

Git 的常用命令

全局变量设置

git config --global user.name "名字"
git config --global user.email "邮箱地址"

初始化 Git 仓库

使用 git init 将一个文件夹初始化为 git 仓库，

git clone

将 GitHub 上的仓库下载到本地电脑上。

git clone 仓库地址

git 提交三部曲

第一步

git add -A

第二步

git commit -m

第三步

git push
初次提交使用：git push origin master

git pull

使用 git pull 进行抓取远程仓库内容，然后进行提交，避免一般的代码冲突

Linux 常用命令

查看当前目录下的文件

ls

查看当前目录下的文件，包括隐藏文件

ls -a

在当前目录下创建文件

touch 文件名称

在当前目录下创建文件夹

mkdir 文件夹名称

进入某个文件夹

cd 文件夹名称

删除当前目录下的指定文件

rm -rf 文件名称
