我的第一篇博客文章

发表于 2020-07-05 更新于 2020-07-06

简介

Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档，它的文件名后缀是.md。

Markdown 语法

  1. 标题

  • 使用 = 和 - 表示一级标题和二级标题

    一级标题
    =======
    二级标题
    -------

    效果：

    一级标题

    二级标题

  • 使用 # 表示标题，共有6级标题

    #一级标题
    ##二级标题
    ###三级标题
    ####四级标题
    #####五级标题
    ######六级标题

    效果:

    一级标题

    二级标题

    三级标题

    四级标题

    五级标题
    六级标题
  1. 段落

段落的前后要有空行，所谓的空行是指没有文字内容。若想在段内强制换行的方式是使用两个以上空格加上回车（引用中换行省略回车）。

  1. 区块引用

在段落的每一行或者第一行使用符号 >，还可以进行多层嵌套引用，例如：

> 区块引用
>> 嵌套引用

效果：

区块引用

嵌套引用

  1. 代码块

代码区块使用四个空格或者一个制表符（Tab 键），例如：

public class HelloWorld {
 public static void main (String[] args) {
  System.out.println (“Hello, World!”);
 }
}

代码块： 

public class HelloWorld {
    public static void main (String[] msg) {
        System.out.println("Hello, World!");
    }
}
  1. 强调

在需要强调的内容两边分别加上 * 或者 _，例如：

*斜体*，_斜体_
**粗体**，__粗体__

效果：

斜体，斜体
粗体，粗体

  1. 列表

使用 * 、+ 、- 表示无序列表，例如：

* 第一行
* 第二行

效果：

  • 第一行
  • 第二行

使用数字 + . 的形式表示有序列表，例如：

1. 第一行
2. 第二行

效果：

  1. 第一行
  2. 第二行
  1. 分割线

使用三个或三个以上的 * 表示下划线，也可用 - 或 _。

  1. 链接

使用 [文字](链接) 的形式表示链接，有两种表示形式：行内式和参考式。

行内式：

[百度](https://www.baidu.com\)。

效果：

百度。

参考式：

[百度][1]
[google][2]

[1]:www.baidu.com
[2]:www.google.com

效果：

百度
google

  1. 图片

添加图片的形式与链接相似，只需要在链接前加上一个 ! 即可。

  1. 反斜杠

反斜杠 \ 相当于转义的作用，即是符号成为可以显示的普通符号。

  1. 标记

使用反引号 ` 可对文字内容进行标记，例如：

`ctrl+a`

效果：

ctrl+a
